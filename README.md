Level-2(task-4)
Lock Authenticator

Obective:

- To develop a secure Lock Authentication System.
- To allow users to register and login using email and password.
- To restrict access to the dashboard only for authenticated users.
- To implement password validation similar to real-world applications like Gmail.

Step Performed:

- Created Register, Login, and Dashboard pages using HTML.
- Designed user-friendly UI using CSS with proper alignment and animations.
- Implemented email validation during registration.
- Added password strength validation (minimum 8 characters).
- Ensured password contains uppercase, lowercase, number, and special character.
- Implemented show/hide password feature
- Stored user credentials securely using localStorage.
- Implemented login authentication logic.
- Displayed error messages for invalid login or registration.
- Restricted dashboard access without login.
- Implemented logout functionality.
- Added option for new user registration after logout.

Tools Used:

- HTML
- CSS
- VISUAL STUDIO CODE
- JAVASCRIPT

Outcome:

- Successfully developed a functional lock authentication system.
- Users can securely register and login.
- Password validation improves security.
- Dashboard is accessible only after authentication.
- Project demonstrates real-time authentication workflow using frontend technologies.
